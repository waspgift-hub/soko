== Description ==
Accept mobile money payments in Tanzania using Mongike API.

Supports:
- M-Pesa
- Airtel Money
- Tigo Pesa
- Halotel

Customers receive a payment prompt on their phone and confirm instantly.

== Installation ==
1. Upload the plugin zip in WordPress
2. Activate plugin
3. Go to WooCommerce > Settings > Payments
4. Enable "Mongike Mobile Money"
5. Enter your API Key
